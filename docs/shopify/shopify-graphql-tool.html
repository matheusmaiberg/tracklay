📖 Tutorial: Como conseguir o Access Token

        
Siga os passos abaixo para criar um app privado na Shopify e obter o token de acesso.


        

          
1

          
Vá em Configurações (Settings) no admin da Shopify.

        

        

          
2

          
Na barra lateral inferior, clique em Apps e canais de vendas.

        

        

          
3

          
Clique em Desenvolver apps (Develop apps). Se não aparecer, você precisa habilitar desenvolvimento privado na loja.

        

        

          
4

          
Clique em Criar app (Create an app). Dê o nome Tracklay Config.

        

        

          
5

          
Clique em Configurar escopos de API do Admin (Configure Admin API scopes).

        

        

          
6

          
Procure por read_shop_metafields e write_shop_metafields. Marque ambos e clique em Salvar.

        

        

          
7

          
Volte para a aba Visão geral (Overview) e clique em Instalar app (Install app).

        

        

          
8

          
Clique em Revelar token uma vez (Reveal token once) para ver o Admin API access token. Copie e cole no campo acima.

        

        

          
9

          
Preencha seu Shop Domain (ex: suevich.myshopify.com) e clique em Gerar script Node.js.

        

        

          
10

          
Salve o arquivo tracklay-setup.js, abra o terminal na pasta e execute: node tracklay-setup.js

        


        

          ⚠️ Importante:
          

            O token tem acesso de leitura e escrita à sua loja. Nunca compartilhe ele. O script roda localmente no seu computador — nenhum dado é enviado para servidores externos.
